# 🏏 IPL Match Winner Predictor (Machine Learning)

A machine learning-based web application that predicts the winner of IPL matches using team details, toss outcome, and venue information. Built with a clean and interactive Streamlit interface.

---

## 🚀 Features

- Predict match winner between two IPL teams  
- Display win probability for both teams  
- Considers toss winner and decision impact  
- Venue-based prediction support  
- Head-to-head match summary  
- Interactive UI with team logos  

---

## 🧠 Machine Learning Model

- **Algorithm:** Logistic Regression  
- **Input Features:**
  - Team 1  
  - Team 2  
  - Toss Winner  
  - Toss Decision (Bat/Field)  
  - Venue  
- **Output:**
  - Predicted Winner  
  - Win Probability (%)  

---

## 🛠️ Tech Stack

- Python  
- Pandas & NumPy  
- Scikit-learn  
- Matplotlib  
- Streamlit  

---

## ⚙️ Installation & Setup

```bash
pip install pandas numpy scikit-learn matplotlib streamlit pillow
pip install -r requirements.txt
▶️ Run the Application
streamlit run Gui.py
```
